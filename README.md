# Simulación


## Como ejecutar
Primero, instalamos las dependencias creando un entorno virtual de python
```bash
python3 -m venv .venv
source .venv/bin/activate
```

Luego, instalamos las dependencias

```bash
pip3 install -r requirements.txt
```

Y listo, ya podemos ejecutar los archivos

Ejemplo:
```
python3 tool-1.py
```
